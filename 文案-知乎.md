# Claude Code 用了几个月，这 7 个 Skill 是真的好用

用了几个月 Claude Code，装过不少 Skill。有的踩坑发现是空壳，有的装上就离不开。整理了一份清单，都是日常编程、写文档、做界面实际在用的。

**开发流程：Superpowers**

从需求到交付一条龙——头脑风暴 → 计划 → TDD → 调试 → 代码审查 → 验证收尾，14 个 skill 协同。写代码时自动介入。装完最大的感受是编码节奏变稳了，少了很多"写到一半发现方向错了"的时刻。

**文档处理：docx、pdf、pptx、xlsx**

直接对话就能生成/编辑 Office 文件。我平时写报告、做 PPT、处理数据表格，以前要切出去开 Office，现在一句话搞定。

**前端设计：frontend-design、ui-ux-pro-max**

frontend-design 官方自带，生成的前端代码没有"AI 味"。ui-ux-pro-max 是升级版，50+ 风格、161 调色板、57 字体搭配，覆盖 React/Vue/Flutter/Tailwind 等十个技术栈。

**加速工具：CodeGraph**

语义级代码地图，给 Claude Code 提供项目全局视野。装了之后工具调用减少约 70%，大型项目里感知明显——不用再反复搜索文件、猜文件位置了。

**自己做的：deepseek-eyes**

让 DeepSeek 也能"看懂"图片的 Skill。把图片发给阿里云百炼的 Qwen 视觉模型，返回中文描述再喂给主模型。用 DeepSeek 的非多模态模型的可以试试。
